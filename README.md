# diary

日记小程序 monorepo。

---

## 产品命名备选（ChatGPT 建议）

我觉得最有潜力的几个（独立开发者做长期品牌）：

### 🥇 言记

理由：

* 和语音天然结合
* 差异化明显
* 可以做 AI 日记

定位：

> 会听你的 AI 日记

---

### 🥈 心迹

理由：

* 比「向心日记」更容易理解
* 品牌感强

---

### 🥉 留白

理由：

* 有高级感
* 适合极简产品

---

### 🏅 向己

理由：

* 有成长路线
* 可以扩展成个人 AI 助手

---

如果结合你的背景（你之前做过「多多录音机」，有音频/转文字方向经验），我其实最推荐：

**「言记」**

产品一句话：

> 说出来，就是你的日记。

入口甚至可以只有一个按钮：

🎙 长按记录今天

AI 自动生成：

* 日记
* 情绪
* 标签
* 回忆卡片

这个方向和普通日记小程序相比，差异会非常明显。

## 快速启动（Phase 0）

前提：本地 Docker 中已有 PostgreSQL（如 `postgres_db`，端口 5432），并已创建库 `diary_dev`。

```bash
# 1. 配置数据库密码（参考 diary-server/.env.example）
export DB_PASSWORD=你的密码

# 2. 启动后端
cd diary-server && ./mvnw spring-boot:run

# 3. 小程序：微信开发者工具导入 diary-wx，关闭域名校验后编译
```

详细说明见各子目录 README。
